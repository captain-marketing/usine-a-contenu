# Scripts vidéos courts

Scripts pour YouTube Shorts, Reels ou TikTok (moins de 90 secondes).

## Convention de nommage

`AAAA-MM-JJ_sujet-court.md`

## Contenu attendu

- Durée cible : 30 à 90 secondes
- Structure : hook (3 sec) → valeur centrale → CTA final
- Format : monologue caméra ou voix-off sur écran
- Écriture pour l'oral : phrases courtes, rythme rapide

## Workflow

1. Idée validée → rédaction avec skill `youtube-short-script`
2. Relecture rythme et clarté
3. Tournage / montage (hors scope Claude)
