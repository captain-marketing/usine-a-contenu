# Scripts vidéos longs

Scripts pour les vidéos YouTube longues (10 minutes et plus).

## Convention de nommage

`AAAA-MM-JJ_titre-video.md`

## Contenu attendu

- Durée cible : 10 à 30 minutes
- Structure : hook → intro + promesse → développement en 3 à 5 parties → conclusion + CTA
- Inclut : indications de coupe, transitions, éléments visuels suggérés
- Écriture pour l'oral : naturel, respirant, sans jargon opaque

## Workflow

1. Angle et plan → validation
2. Rédaction complète avec skill `youtube-long-video-script`
3. Relecture fluidité et durée estimée
4. Tournage / montage (hors scope Claude)
