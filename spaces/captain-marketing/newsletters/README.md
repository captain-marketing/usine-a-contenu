# Newsletters

Éditions de la newsletter **SYSTEMIA** envoyée via Ghost.

## Convention de nommage

`AAAA-MM-JJ_systemia-nXXX.md`

## Contenu attendu

- Format : édition hebdomadaire ou bimensuelle
- Structure type : accroche + 1 sujet principal + ressources sélectionnées + CTA
- Ton : direct, personnel, conversationnel
- Longueur : 600 à 1 200 mots

## Workflow

1. Angle + brief → validation manuelle
2. Rédaction → `usine-a-contenu:produce`
3. Relecture style → skill `style-captain-marketing`
4. Envoi → Ghost newsletter
