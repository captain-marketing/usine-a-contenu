# Posts Instagram

Contenus publiés sur le compte Instagram Captain Marketing.

## Convention de nommage

`AAAA-MM-JJ_sujet-court.md`

## Contenu attendu

- Format : carrousel (recommandé), post image ou Reel
- Légende : accroche + corps + hashtags (10 à 20 max)
- Visuels à stocker dans `../assets/`
- Ton : accessible, visuel, éducatif

## Workflow

1. Concept visuel + texte → validation
2. Création visuel → Canva ou assets existants
3. Rédaction légende → repurposing ou production dédiée
4. Publication ou programmation → Zernio (MCP disponible)
