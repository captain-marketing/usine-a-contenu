# Articles de blog

Articles publiés sur [blog.captainmarketing.io](https://blog.captainmarketing.io) via Ghost CMS.

## Convention de nommage

`AAAA-MM-JJ_slug-de-larticle.md`

## Contenu attendu

- Articles longs (1 500 à 3 000 mots)
- Formats : tutoriels, guides pratiques, analyses, études de cas
- Optimisés SEO avec métadonnées Ghost (voir skill `usine-a-contenu:ghost-seo`)
- Ton : pédagogique, concret, actionnable

## Workflow

1. Idéation → `usine-a-contenu:ideation`
2. Rédaction → `usine-a-contenu:produce`
3. Optimisation SEO → `usine-a-contenu:optimize`
4. Métadonnées Ghost → `usine-a-contenu:ghost-seo`
5. Publication → Ghost CMS
