# Assets

Ressources visuelles et médias utilisés dans les contenus Captain Marketing.

## Organisation suggérée

```
assets/
├── images/        # Visuels pour articles, LinkedIn, Instagram
├── covers/        # Images de couverture Ghost (1 200 x 630 px recommandé)
├── thumbnails/    # Miniatures YouTube
└── logos/         # Logos et éléments de branding
```

## Formats acceptés

- Images : JPG, PNG, WebP
- Vidéos légères : MP4 (extraits courts uniquement, les fichiers lourds restent sur disque local)
- Documents : PDF

## Conventions

- Nommage : `sujet-type-format.ext` (ex. `ia-marketing-cover-blog.jpg`)
- Résolution minimale : 1 200 px de large pour le web
- Compression systématique avant upload Ghost ou réseaux sociaux
