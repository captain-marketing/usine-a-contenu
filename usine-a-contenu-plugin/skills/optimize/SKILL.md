## Étape 2 — Audit rapide

```
## Audit SEO/GEO — [Titre]
1. Mot-clé principal : [présent / absent / mal positionné]
```

## Étape 3 — Version optimisée

**SEO :** mot-clé dans titre + premier paragraphe + au moins un H2, balises méta, maillage interne depuis MEMORY.md.

**GEO :** réponse directe dans les 2-3 premières phrases, formulations naturelles (comment/pourquoi/qu'est-ce que), données factuelles, FAQ si pertinent.

5-7 points concrets sur les changements effectués.
